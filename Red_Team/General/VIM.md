Vim  i (insert mode) esc (exit mode)  https://vimsheet.com/
```

x Cut character   (add # before any cmd to multiply)
dw Cut word
dd Cut full line
yw Copy word
yy Copy full line
p Paste
:1 Go to line number 1.
:w Write the file, save
:q Quit
:q! Quit without saving
:wq Write and quit
```
